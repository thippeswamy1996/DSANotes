### What is a Constructor?

In **Java**, a **constructor** is a special method that is automatically called when an object of a class is created. Its primary purpose is to **initialize objects**.

#### Key Characteristics:

* The constructor name **must match** the class name.
* It **does not have a return type**, not even `void`.
* Java provides a **default constructor** if no constructors are defined.

#### Example:

```java
public class Car {
    String model;

    // Constructor
    public Car(String model) {
        this.model = model;
    }
}
```

Usage:

```java
Car myCar = new Car("Toyota");
```

---

### Application of Constructor in Selenium with Java

In **Selenium Automation Frameworks** (like Page Object Model - POM), constructors are commonly used for:

#### 1. **Initializing WebDriver**

You typically pass the `WebDriver` instance to page classes using a constructor.

#### Example:

```java
// LoginPage.java
public class LoginPage {
    WebDriver driver;

    // Constructor
    public LoginPage(WebDriver driver) {
        this.driver = driver;
        PageFactory.initElements(driver, this);  // Initializes WebElements
    }

    @FindBy(id = "username")
    WebElement usernameInput;

    public void enterUsername(String username) {
        usernameInput.sendKeys(username);
    }
}
```

#### 2. **Page Object Model (POM) Design**

In POM, each page of the application is represented as a Java class. Constructors are used to:

* Accept and share the `WebDriver` instance.
* Initialize page elements.
* Promote reusability and maintainability.

#### 3. **Framework Scalability**

Using constructors makes the framework **more modular**, where you can easily create and test different page objects without redefining WebDriver every time.

---

### Summary:

| Concept     | Role in Java                             | Role in Selenium                              |
| ----------- | ---------------------------------------- | --------------------------------------------- |
| Constructor | Initializes objects                      | Initializes page objects with WebDriver       |
| Application | Sets up default values or assigns values | Helps in POM, reusability, and initialization |
